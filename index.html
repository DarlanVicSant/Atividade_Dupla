Atividade Dupla

Essa é a atividade dupla, onde o aluno irá praticar os conceitos de HTML e CSS.

HTML

O HTML é a linguagem de marcação utilizada para estruturar o conteúdo da página. Ele é composto por elementos, que são representados por tags. As tags são usadas para definir a estrutura e o conteúdo da página.

CSS

O CSS é a linguagem de estilo utilizada para definir a aparência da página. Ele é composto por regras, que são formadas por um seletor e um bloco de declarações. O seletor é usado para selecionar os elementos que serão estilizados, e o bloco de declarações é onde são definidas as propriedades e valores de estilo.
